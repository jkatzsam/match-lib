matchlib
========

matchlib is a Python module for string-matching and distributed
under the 3-Clause BSD license. 

The project started in 2013 by Russell Funk and Julian Katz-Samuels.

It is currently maintained by Russell Funk and Julian Katz-Samuels.

Dependencies
============

Matchlib is tested to work on python 2.7. 

There are no required dependencies to build software. 


Install
=======

This package uses distutils, which is the default way of installing
python modules. To install in your home directory, use::

  python setup.py install --user

To install for all users on Unix/Linux::

  python setup.py build
  sudo python setup.py install
